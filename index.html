Librerias Utilizadas

Open CV2

Es una biblioteca de visión por computadora de código abierto ampliamente utilizada. OpenCV proporciona una amplia gama de funciones y algoritmos optimizados para el procesamiento de imágenes y videos.

MediaPipe

MediaPipe es una biblioteca de código abierto desarrollada por Google que se utiliza para construir aplicaciones de percepción y análisis en tiempo real. Proporciona una amplia gama de soluciones y herramientas para el procesamiento de datos multimodales, como imágenes, video, audio y señales 3D.

PyAutoGUI

Es una biblioteca de Python que proporciona funciones para controlar y automatizar la interacción con la interfaz gráfica de usuario (GUI) de una computadora. PyAutoGUI permite realizar acciones como mover el mouse, hacer clic en elementos de la pantalla, escribir texto, capturar imágenes de la pantalla y realizar otras operaciones relacionadas con la automatización de tareas en el escritorio.


Vision artificial

1Que es la vision artificial

La visión artificial es una disciplina científica que persigue que los ordenadores alcancen esa misma capacidad, es decir, trata de conseguir que las máquinas puedan percibir y comprender una o varias imágenes y actuar de una manera determinada..

2Que es un mouse virtual con vision artificial

El proyecto consiste en desarrollar un mouse virtual basado en visión artificial, que permita a los usuarios interactuar con un sistema informático sin necesidad de un dispositivo físico como un mouse convencional. En lugar de eso, el mouse virtual utilizará técnicas de visión artificial para detectar y rastrear los movimientos de las manos y los gestos realizados por el usuario.
